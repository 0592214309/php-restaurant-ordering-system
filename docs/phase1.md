# Phase 1: Project Setup & Database Design

## How to Set Up (Beginner Friendly)

1. **Install XAMPP**
   - Download and install XAMPP from [apachefriends.org](https://www.apachefriends.org/index.html)
   - Start Apache and MySQL from XAMPP Control Panel

2. **Import the Database**
   - Open phpMyAdmin: [http://localhost/phpmyadmin](http://localhost/phpmyadmin)
   - Click "Import" and select `database/restaurant.sql`
   - Click "Go" to create tables and sample data

3. **Project Files**
   - Main files:
     - `config/db.php` (database connection)
     - `index.php` (homepage/menu)
     - `assets/css/style.css` (baby blue theme)
     - `assets/js/main.js` (simple JS)

4. **Run the App**
   - Go to [http://localhost/restaurant/index.php](http://localhost/restaurant/index.php)
   - You should see the menu items and a nice blue UI
